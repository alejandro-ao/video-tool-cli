# Context Cards Template

Extract moments from transcript where speaker references other content.

## Output Format

### YouTube Card Suggestions
- HH:MM:SS — [Brief note why this needs a card]. [Quote or paraphrase]

### Resource Mentions
- HH:MM:SS — [Description of resource and why mentioned]

## Guidelines
- Only use timestamps that appear in the VTT transcript
- Card suggestions: moments referencing other videos by the speaker
- Resource mentions: external links, papers, tools, repos, products
- If none found, write "None detected in the transcript."
